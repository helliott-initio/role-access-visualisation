# Requirements: Role Access Visualisation

**Defined:** 2026-01-27
**Core Value:** A flexible flowchart editor that just works

## v1 Requirements

Requirements for this milestone. Each maps to roadmap phases.

### Canvas & Connectors

- [ ] **CANVAS-01**: Edge connects to the exact handle user drags to (not snap to default)
- [ ] **CANVAS-02**: Visual feedback shows valid connection targets when dragging
- [ ] **CANVAS-03**: Reconnecting an edge preserves handle positions

### Project Management

- [ ] **PROJ-01**: App launches to "New Project" screen (not demo data)
- [ ] **PROJ-02**: User can create blank project with name
- [ ] **PROJ-03**: User can rename existing project tab
- [ ] **PROJ-04**: User can delete project tab with confirmation

### Editor Features

- [ ] **EDIT-01**: Undo reverses last action
- [ ] **EDIT-02**: Redo restores undone action
- [ ] **EDIT-03**: Undo/redo works for all edit operations (add, delete, move, connect)

### UX Polish

- [ ] **UX-01**: Clear visual cue for how to add new groups
- [ ] **UX-02**: Clear visual cue for how to connect nodes
- [ ] **UX-03**: Empty project shows helpful onboarding hints

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Advanced Features

- **ADV-01**: Keyboard shortcuts for common actions (delete, undo, redo)
- **ADV-02**: Search/filter nodes by label or email
- **ADV-03**: Minimap for large diagrams
- **ADV-04**: Copy/paste nodes between projects

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multi-user collaboration | Personal tool, complexity not justified |
| Cloud storage/auth | localStorage sufficient for personal use |
| Mobile/touch support | Desktop-first editor tool |
| Real-time sync | Single user, no need |
